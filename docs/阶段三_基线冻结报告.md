## 4. 选定 eps 的判据

使用统一 envelope：

- `short cross-cluster P95 <= 25%`
- `medium cross-cluster P95 <= 10%`
- `long cross-cluster P95 <= 10%`
- `short same-cluster rate <= 10%`
- `component_count = 1`
- `isolate_count = 0`
- `short no_path_rate <= 1%`

在满足上述约束的候选中，选择压缩率最高的 `eps`。
