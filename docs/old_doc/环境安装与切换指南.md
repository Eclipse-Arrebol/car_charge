# 环境安装与切换指南

本文档说明如何在 Linux / 云主机上安装并切换 `RTX 4090` 与 `RTX 5090` 两套运行环境。

## 1. 前提

进入项目根目录：

```bash
cd car_charge
```

如果你是第一次拉代码：

```bash
git clone https://github.com/Eclipse-Arrebol/car_charge.git
cd car_charge
```

## 2. 安装 4090 环境

执行：

```bash
bash script/install_env_4090.sh
```

默认安装位置：

```bash
/root/.venvs/ev-demo-4090
```

激活环境：

```bash
source /root/.venvs/ev-demo-4090/bin/activate
```

默认配置：

- PyTorch: `2.3.1`
- CUDA: `cu118`

验证：

```bash
python -c "import torch; print(torch.__version__); print(torch.version.cuda); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
```

## 3. 安装 5090 环境

执行：

```bash
bash script/install_env_5090.sh
```

默认安装位置：

```bash
/root/.venvs/ev-demo-5090
```

激活环境：

```bash
source /root/.venvs/ev-demo-5090/bin/activate
```

默认配置：

- PyTorch: `2.7.1`
- CUDA: `cu128`

验证：

```bash
python -c "import torch; print(torch.__version__); print(torch.version.cuda); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
```

## 4. 环境切换

切到 4090 环境：

```bash
source /root/.venvs/ev-demo-4090/bin/activate
```

切到 5090 环境：

```bash
source /root/.venvs/ev-demo-5090/bin/activate
```

退出当前虚拟环境：

```bash
deactivate
```

## 5. 常用训练命令

激活对应环境后，在项目根目录执行：

小规模快速验证：

```bash
python main.py train-real --debug
python main.py evaluate --debug
```

中等规模验证：

```bash
python main.py train-real --medium
python main.py evaluate --medium
```

完整规模训练：

```bash
python main.py train-real
python main.py evaluate
```

## 6. 说明

- `--debug` 适合快速检查逻辑是否正常。
- `--medium` 适合中等规模验证训练效果。
- 不带参数时使用完整规模配置。
- 如果显卡与环境不匹配，优先检查 `torch.version.cuda`、`torch.cuda.is_available()` 和显卡驱动版本。
