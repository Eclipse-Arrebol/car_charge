# 阶段三层级路网形式化初稿

## 1. 写作目标

将当前“真实图 `L0` -> 训练图 `L1`”方案从工程实现提升为论文中可直接引用的形式化定义，供方法章节使用。

---

## 2. 符号约定

- `G_{L0} = (V_{L0}, E_{L0}, \ell_0)`：真实可通行路网
- `G_{L1} = (V_{L1}, E_{L1}, \ell_1)`：训练决策图
- `\ell_0, \ell_1`：边长度函数
- `cluster(u)`：`u \in V_{L0}` 所属 cluster
- `r(cluster)`：该 cluster 的代表锚点
- `\pi : V_{L0} \to V_{L1}`：节点映射
- `\rho : E_{L1} \to \mathrm{Path}(G_{L0})`：边展开映射

当前方案中，`V_{L1} \subseteq V_{L0}`，因为 `L1` 节点使用 `L0` 中的代表锚点，而不是虚拟几何质心。

## 3. LaTeX 初稿

```latex
\paragraph{Hierarchical Road Network Construction.}
We construct a two-level hierarchical road representation
$\mathcal{H}=(G_{L0}, G_{L1}, \pi, \rho)$ from the real drivable road network.
The lower-level graph $G_{L0}=(V_{L0}, E_{L0}, \ell_0)$ denotes the real
drivable road network, where $\ell_0:E_{L0}\rightarrow\mathbb{R}_+$ is the
edge-length function.
The upper-level graph $G_{L1}=(V_{L1}, E_{L1}, \ell_1)$ denotes the
decision-level training graph used by the GNN policy.

\paragraph{Node Abstraction.}
The node set $V_{L1}$ is obtained by clustering nearby road nodes under a
distance threshold $\varepsilon$ and a local topological path constraint.
For each cluster $C$, we select a representative anchor node
$r(C)\in V_{L0}$, and define the training node as this representative node.
Therefore, $V_{L1}\subseteq V_{L0}$ holds by construction.
The node projection is defined as
\[
\pi(u)=r(\mathrm{cluster}(u)), \qquad \forall u\in V_{L0}.
\]
For every representative node $v\in V_{L1}$, we have $\pi(v)=v$.

\paragraph{Edge Abstraction.}
Each edge $e\in E_{L1}$ corresponds to a real path in $G_{L0}$ connecting the
two representative anchors of its incident clusters.
We denote this path lifting by
\[
\rho:E_{L1}\rightarrow \mathrm{Path}(G_{L0}).
\]
The decision-level edge length is defined by the real path length, i.e.,
\[
\ell_1(e)=\sum_{e'\in \rho(e)} \ell_0(e'), \qquad \forall e\in E_{L1}.
\]
Hence, every training edge preserves the physical travel distance of its
corresponding real road path.

\paragraph{Fidelity Metric.}
To quantify the abstraction quality, for any pair $(u,v)\in V_{L0}\times V_{L0}$
with $\pi(u)\neq \pi(v)$, we define the relative shortest-path distortion as
\[
\mathrm{err}(u,v)=
\frac{\left|d_{L1}(\pi(u),\pi(v)) - d_{L0}(u,v)\right|}
{d_{L0}(u,v)},
\]
where $d_{L0}$ and $d_{L1}$ are weighted shortest-path distances on
$G_{L0}$ and $G_{L1}$, respectively.

\paragraph{Empirical Fidelity Envelope.}
We evaluate the abstraction by stratified sampling over short-, medium-, and
long-range node pairs.
With $\varepsilon=40\,$m, the resulting abstraction satisfies the empirical
fidelity envelope:
short-range cross-cluster P95 $\le 25\%$,
medium-range cross-cluster P95 $\le 10\%$,
long-range cross-cluster P95 $\le 10\%$,
short-range same-cluster rate $\le 10\%$,
while maintaining a single connected component and zero isolated nodes in
$G_{L1}$.
```

## 4. 中文解释版

可以在论文写作时按下面的逻辑展开：

1. `L0` 是真实可通行路网，用于真实路径和代价计算
2. `L1` 是决策层训练图，用于 GNN 学习
3. `L1` 节点不是虚拟质心，而是 cluster 的代表锚点，所以 `V_{L1} \subseteq V_{L0}`
4. `\pi` 负责把 `L0` 节点映射到其所在 cluster 的代表锚点
5. `\rho` 负责把 `L1` 边展开回 `L0` 中的一条真实路径
6. `\ell_1` 不用欧氏距离，而是直接继承 `\rho(e)` 的真实路径长度
7. fidelity 用 `L0/L1` 最短路差异来衡量，并通过分层采样给出 empirical envelope
