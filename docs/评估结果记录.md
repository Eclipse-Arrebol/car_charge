## 当前架构说明

### GraphQNetwork（第三轮起）

```
原始节点特征 [N, 15]
    ↓  FeatureEncoder
      ├─ 队列/容量组 [1,2,4,5,14]  → Linear(5→32) → ReLU
      ├─ 价格/时间组 [3,7,10,11,12,13] → Linear(6→32) → ReLU
      ├─ 空间/电网组 [0,6,9]        → Linear(3→32) → ReLU
      └─ EV 状态组  [8]             → Linear(1→32) → ReLU
      → concat [128] → Linear(128→64) → LayerNorm → ReLU
编码特征 [N, 64]
    ↓  GATv2Conv(64→32, heads=4)
    ↓  GATv2Conv(32→64, heads=4)
    ↓  global_mean_pool → global_ctx [B, 64]
    ↓  Dueling Head
      ├─ Value stream:     global_ctx → V(s) [B, 1]
      └─ Advantage stream: station_emb + global_ctx → A(s,a) [B, 1] × num_stations
Q = V(s) + A(s,a) − mean_a(A)  →  Action Mask  →  输出
```
